# algoritmohanne

Menus interativos que permitem manipular as matrizes para um conjunto pré-determinado de operações. A interface de menu proposta é apresentada a seguir, mostrando uma sessão de interação com o usuário. Uma vez realizada uma operação do menu, ele é reapresentado para o usuário.

* Tamanho máximo que a matriz pode ter de linhas e colunas: 10
* Para visualizar a matriz é necessário digitar a opção (10), (11) ou (12) para imprimir respectivamente A,B,C.
